# 🛒 Smart Cart - E-Commerce Website

Smart Cart is a responsive and user-friendly **E-Commerce Website** designed to showcase products and provide a smooth shopping experience. It includes product listings, add-to-cart functionality, and a modern UI.

---

## 🌐 Live Demo

👉 **Click below to open website directly:**

🔗 https://arpita2911.github.io/Smart-Cart---E-Commerce-Site/

---

## 📌 Features

* 🛍️ Product listing with images
* 🛒 Add to Cart functionality
* 📱 Fully responsive design
* ⚡ Fast and lightweight
* 🎨 Clean and modern UI
* 🔔 Interactive user experience

---

## 🛠️ Technologies Used

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/arpita2911/Smart-Cart---E-Commerce-Site.git
```

2. Open folder
3. Run `index.html` in browser
